# 初始化项目记录

## 创建项目

```bash
npx create-next-app@latest .
```

## 安装依赖

```bash
pnpm install
```

## 启动项目

```bash
pnpm dev
```
# 依赖安装
第三方内容编辑器tiatap
## 表格

| 包名 | 作用 |
|------|------|
| @tiptap/react | React 绑定 |
| @tiptap/starter-kit | 基础扩展（标题、段落、加粗、列表等） |
| @tiptap/extension-image | 图片插入 |
| @tiptap/extension-placeholder | 占位符提示 |
| @tiptap/extension-table | 表格支持 |
| @tiptap/extension-task-item | 任务清单项 |
| @tiptap/extension-task-list | 任务清单列表 |

## 安装命令

```bash
pnpm add @tiptap/react @tiptap/starter-kit @tiptap/extension-image @tiptap/extension-placeholder @tiptap/extension-table @tiptap/extension-task-item @tiptap/extension-task-list
```
