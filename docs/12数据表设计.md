## 二、完整 DDL

```sql
-- ============================================================
-- Phase 0: 扩展
-- ============================================================
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";     -- UUID 生成
CREATE EXTENSION IF NOT EXISTS pg_trgm;          -- 模糊搜索/相似度
CREATE EXTENSION IF NOT EXISTS pgcrypto;         -- 密码哈希 (bcrypt)

-- ============================================================
-- 枚举类型
-- ============================================================
CREATE TYPE node_type     AS ENUM ('folder', 'file');
CREATE TYPE membership    AS ENUM ('free', 'pro', 'enterprise');
CREATE TYPE device_type   AS ENUM ('desktop', 'mobile', 'tablet', 'web');
CREATE TYPE session_status AS ENUM ('active', 'expired', 'revoked');

-- ============================================================
-- 1. users — 用户账户
-- ============================================================
CREATE TABLE users (
    id            UUID            PRIMARY KEY DEFAULT uuid_generate_v4(),
    uid           VARCHAR(32)     NOT NULL UNIQUE,                 -- 公开 ID，如 'A3fG9kL2'
    email         VARCHAR(255)    NOT NULL UNIQUE,
    phone         VARCHAR(20),
    full_name     VARCHAR(100)    NOT NULL,
    avatar        VARCHAR(1)      NOT NULL DEFAULT '用',            -- 首字作为头像
    membership    membership      NOT NULL DEFAULT 'free',
    password_hash VARCHAR(255)    NOT NULL,                         -- bcrypt hash

    created_at    TIMESTAMPTZ     NOT NULL DEFAULT now(),
    updated_at    TIMESTAMPTZ     NOT NULL DEFAULT now()
);

CREATE INDEX idx_users_email ON users(email);

-- ============================================================
-- 2. document_nodes — 文件树节点
--    覆盖前端类型：TreeNode + TrashItemData
-- ============================================================
CREATE TABLE document_nodes (
    -- 主键
    id            TEXT            PRIMARY KEY,  -- 'folder-1', 'file-1-2-3'

    -- 用户隔离
    user_id       UUID            NOT NULL REFERENCES users(id) ON DELETE CASCADE,

    -- 邻接表
    parent_id     TEXT            REFERENCES document_nodes(id) ON DELETE SET NULL,

    -- 物化路径（如 'folder-1.folder-1-2.file-1-2-3'）
    path          VARCHAR(2048)   NOT NULL DEFAULT '',

    -- 核心属性
    name          VARCHAR(255)    NOT NULL DEFAULT '未命名',
    type          node_type       NOT NULL,
    level         SMALLINT        NOT NULL DEFAULT 0,               -- 冗余，物化路径可推导
    order_index   INT             NOT NULL DEFAULT 0,               -- 同级排序

    -- 文件内容（仅 type='file' 有值）
    content       TEXT,
    word_count    INT             DEFAULT 0,

    -- 时间戳
    created_at    TIMESTAMPTZ     NOT NULL DEFAULT now(),
    updated_at    TIMESTAMPTZ     NOT NULL DEFAULT now(),

    -- 软删除（回收站：deleted_at IS NOT NULL）
    deleted_at    TIMESTAMPTZ
);

-- 用户活跃节点：获取完整树
CREATE INDEX idx_nodes_user_active
    ON document_nodes(user_id) WHERE deleted_at IS NULL;

-- 直接子节点查询
CREATE INDEX idx_nodes_parent
    ON document_nodes(parent_id, order_index) WHERE deleted_at IS NULL;

-- 物化路径前缀：子树查询
CREATE INDEX idx_nodes_path
    ON document_nodes(user_id, path text_pattern_ops) WHERE deleted_at IS NULL;

-- 回收站列表
CREATE INDEX idx_nodes_trash
    ON document_nodes(user_id, deleted_at) WHERE deleted_at IS NOT NULL;

-- 全文搜索（可选 Phase 3）
-- ALTER TABLE document_nodes ADD COLUMN search_vector tsvector
--     GENERATED ALWAYS AS (to_tsvector('simple', name || ' ' || content)) STORED;
-- CREATE INDEX idx_nodes_search ON document_nodes USING GIN(search_vector);

-- ============================================================
-- 3. user_sessions — 登录会话
--    替代前端 localStorage/sessionStorage
-- ============================================================
CREATE TABLE user_sessions (
    id            UUID            PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id       UUID            NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    token_hash    VARCHAR(64)     NOT NULL UNIQUE,                  -- SHA-256(token)
    device_name   VARCHAR(255),                                     -- 'Windows 桌面端'
    device_type   device_type     DEFAULT 'web',
    ip_address    INET,
    user_agent    TEXT,
    status        session_status  NOT NULL DEFAULT 'active',
    created_at    TIMESTAMPTZ     NOT NULL DEFAULT now(),
    expires_at    TIMESTAMPTZ     NOT NULL,                         -- rememberMe=true 更长
    revoked_at    TIMESTAMPTZ
);

CREATE INDEX idx_sessions_user   ON user_sessions(user_id, status);
CREATE INDEX idx_sessions_token  ON user_sessions(token_hash);

-- ============================================================
-- 4. user_notification_settings — 通知偏好
--    覆盖：NotificationsTab
-- ============================================================
CREATE TABLE user_notification_settings (
    user_id                UUID     PRIMARY KEY REFERENCES users(id) ON DELETE CASCADE,
    document_update        BOOLEAN  NOT NULL DEFAULT true,         -- 文档更新通知
    comment_feedback       BOOLEAN  NOT NULL DEFAULT true,         -- 评论与反馈
    email_digest           BOOLEAN  NOT NULL DEFAULT false,        -- 邮件摘要（每日/周）
    sound_enabled          BOOLEAN  NOT NULL DEFAULT true,         -- 声音提示
    updated_at             TIMESTAMPTZ NOT NULL DEFAULT now()
);

-- ============================================================
-- 5. user_devices — 已登录设备
--    覆盖：DevicesTab
-- ============================================================
CREATE TABLE user_devices (
    id            UUID            PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id       UUID            NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    session_id    UUID            REFERENCES user_sessions(id) ON DELETE SET NULL,
    device_name   VARCHAR(255)    NOT NULL,
    device_type   device_type     NOT NULL DEFAULT 'web',
    os_info       VARCHAR(255),
    ip_address    INET,
    last_active   TIMESTAMPTZ     NOT NULL DEFAULT now(),
    is_current    BOOLEAN         NOT NULL DEFAULT false,          -- 是否当前设备
    is_trusted    BOOLEAN         NOT NULL DEFAULT false,          -- 是否信任设备
    created_at    TIMESTAMPTZ     NOT NULL DEFAULT now()
);

CREATE UNIQUE INDEX idx_devices_current
    ON user_devices(user_id) WHERE is_current = true;              -- 每用户最多一个当前设备

CREATE INDEX idx_devices_user ON user_devices(user_id, last_active DESC);

-- ============================================================
-- 6. user_recent_searches — 最近搜索记录
--    覆盖：SearchResultsView 中的 recentSearches
-- ============================================================
CREATE TABLE user_recent_searches (
    id            UUID            PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id       UUID            NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    term          VARCHAR(255)    NOT NULL,
    searched_at   TIMESTAMPTZ     NOT NULL DEFAULT now()
);

CREATE INDEX idx_searches_user ON user_recent_searches(user_id, searched_at DESC);

-- 每个用户最多保留 10 条记录（触发器清理）
CREATE OR REPLACE FUNCTION trim_recent_searches()
RETURNS TRIGGER AS $$
BEGIN
    DELETE FROM user_recent_searches
    WHERE user_id = NEW.user_id
      AND id NOT IN (
          SELECT id FROM user_recent_searches
          WHERE user_id = NEW.user_id
          ORDER BY searched_at DESC
          LIMIT 9
      );
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER trg_trim_searches
    AFTER INSERT ON user_recent_searches
    FOR EACH ROW EXECUTE FUNCTION trim_recent_searches();

-- ============================================================
-- 7. membership_plans — 会员套餐（配置表，非用户数据）
-- ============================================================
CREATE TABLE membership_plans (
    id            SMALLINT        PRIMARY KEY,                     -- 1=free, 2=pro, 3=team
    plan_key      VARCHAR(20)     NOT NULL UNIQUE,                 -- 'free', 'pro', 'enterprise'
    name          VARCHAR(50)     NOT NULL,                        -- '免费版', '专业版', '团队版'
    price_cents   INT             NOT NULL DEFAULT 0,              -- 价格（分），0=免费
    period        VARCHAR(10)     NOT NULL DEFAULT 'month',        -- 'month' / 'year'
    description   TEXT,
    features      JSONB           NOT NULL DEFAULT '[]',           -- 功能列表 JSON 数组
    storage_mb    INT             NOT NULL DEFAULT 100,            -- 存储上限（MB）
    max_spaces    INT             NOT NULL DEFAULT 3,              -- 文档空间上限（-1=无限）
    highlighted   BOOLEAN         NOT NULL DEFAULT false,          -- 是否推荐
    is_active     BOOLEAN         NOT NULL DEFAULT true
);
```

## 三、关键操作 SQL（参考实现）

### 3.1 用户注册

```sql
INSERT INTO users (uid, email, full_name, password_hash)
VALUES ($1, $2, $3, crypt($4, gen_salt('bf')));
```

### 3.2 用户登录（密码验证）

```sql
SELECT id, uid, email, full_name, avatar, membership, created_at
FROM users
WHERE email = $1 AND password_hash = crypt($2, password_hash);
```

### 3.3 创建登录会话

```sql
INSERT INTO user_sessions (user_id, token_hash, device_name, device_type, ip_address, user_agent, expires_at)
VALUES ($1, encode(sha256($2::bytea), 'hex'), $3, $4, $5, $6, $7);

-- 同步更新设备列表
INSERT INTO user_devices (user_id, session_id, device_name, device_type, ip_address, last_active, is_current)
VALUES ($1, (SELECT id FROM user_sessions WHERE token_hash = $hash), $3, $4, $5, now(), true)
ON CONFLICT ON CONSTRAINT idx_devices_uniq_device
DO UPDATE SET session_id = EXCLUDED.session_id, last_active = now(), is_current = true;
```

### 3.3" 撤销其他设备

```sql
UPDATE user_devices SET is_current = false
WHERE user_id = $1 AND id != $current_device_id;

-- 撤销指定设备
UPDATE user_sessions SET status = 'revoked', revoked_at = now()
WHERE id = (SELECT session_id FROM user_devices WHERE id = $device_id);
```

### 3.4 获取通知偏好

```sql
SELECT * FROM user_notification_settings WHERE user_id = $1;
```

初次访问时自动创建默认行：

```sql
INSERT INTO user_notification_settings (user_id)
VALUES ($1)
ON CONFLICT (user_id) DO NOTHING;
```

### 3.5 更新通知偏好

```sql
UPDATE user_notification_settings
SET document_update = $2, comment_feedback = $3, sound_enabled = $4, updated_at = now()
WHERE user_id = $1;
```

### 3.6 记录搜索历史

```sql
INSERT INTO user_recent_searches (user_id, term)
VALUES ($1, $2)
ON CONFLICT DO NOTHING;
-- 触发器自动保留最近 10 条
```

### 3.7 获取最近搜索

```sql
SELECT term FROM user_recent_searches
WHERE user_id = $1
ORDER BY searched_at DESC
LIMIT 10;
```

### 3.8 获取文档树完整操作（已在 12 节详述，此处摘要）

| 操作 | SQL 模式 |
|------|----------|
| 获取用户完整树 | `SELECT * FROM document_nodes WHERE user_id=$1 AND deleted_at IS NULL ORDER BY path, order_index` |
| 获取直接子节点 | `SELECT * FROM document_nodes WHERE parent_id=$1 AND deleted_at IS NULL ORDER BY order_index` |
| 搜索所有子孙 | `SELECT * FROM document_nodes WHERE user_id=$1 AND path LIKE $2\\|\|'.%' AND deleted_at IS NULL` |
| 移动节点（事务） | `UPDATE` 自身 + `UPDATE` 子孙 `path LIKE old_path\|\|'.%'` |
| 软删除 | `UPDATE document_nodes SET deleted_at=now() WHERE id=$1 OR path LIKE $path\|\|'.%'` |
| 恢复 | `UPDATE document_nodes SET deleted_at=NULL WHERE id=$1 OR path LIKE $path\|\|'.%'` |
| 永久删除 | `DELETE FROM document_nodes WHERE deleted_at IS NOT NULL AND ...` |

### 3.9 全文搜索（PostgreSQL 全文）

```sql
-- 使用 tsvector + tsquery
SELECT id, name, type, path, ts_headline('simple', content, query) AS snippet
FROM document_nodes, plainto_tsquery('simple', $query) AS query
WHERE user_id = $user_id
  AND search_vector @@ query
  AND deleted_at IS NULL
  AND type = 'file'
ORDER BY ts_rank(search_vector, query) DESC;
```

## 四、前端类型 → 数据库列映射

### 4.1 users ↔ `User`

| 前端 `User` | DB `users` | 说明 |
|------------|------------|------|
| `id` | `id` | UUID |
| `uid` | `uid` | 公开标识 |
| `email` | `email` | UNIQUE |
| `phone?` | `phone` | 可选 |
| `fullName` | `full_name` | snake_case |
| `avatar` | `avatar` | 首字字符 |
| `membership` | `membership` | enum 映射 |
| `createdAt` | `created_at` | ISO 8601 ↔ timestamptz |
| （无） | `password_hash` | bcrypt，前端永远不返回 |
| （无） | `updated_at` | 仅服务端维护 |

**注意**：前端 `User` 类型不包含 `password`，密码通过独立 API 传输，哈希后存入 `password_hash`，前端永远不会持有哈希。

### 4.2 document_nodes ↔ `TreeNode` + `TrashItemData`

| 前端 `TreeNode` | DB `document_nodes` | 说明 |
|----------------|---------------------|------|
| `id` | `id` | 直接映射 |
| `name` | `name` | 直接映射 |
| `type` | `type` | `'folder'` / `'file'` |
| `level` | `level` | 直接映射 |
| `children` | 通过 `parent_id` / `path` 查询 | 应用层组装 |
| `expanded` | 不存储 | 前端状态 |
| `createdAt?` | `created_at` | 时间戳格式化 |
| `updatedAt?` | `updated_at` | 时间戳格式化 |
| `wordCount?` | `word_count` | 直接映射 |
| `content?` | `content` | 仅 file 有值 |

| 前端 `TrashItemData` | DB `document_nodes` | 说明 |
|----------------------|---------------------|------|
| `id` | `id` | `WHERE deleted_at IS NOT NULL` |
| `type` | `type` | 直接映射 |
| `name` | `name` | 直接映射 |
| `count?` | 子查询 | `COUNT(*) WHERE path LIKE ...` |
| `lastModified` | `updated_at` | 直接映射 |
| `createdAt` | `created_at` | 直接映射 |
| `content?` | `content` | 直接映射 |
| `wordCount?` | `word_count` | 直接映射 |

### 4.3 user_sessions ↔ 登录状态

| 概念 | 当前实现 | DB 方案 |
|------|---------|---------|
| 会话存储 | `localStorage` / `sessionStorage` (JSON) | `user_sessions` 表 |
| `rememberMe` | `true`→localStorage, `false`→sessionStorage | `expires_at`：1年 vs 24小时 |
| 登出 | `localStorage.removeItem()` | `UPDATE status='revoked'` |
| 注销（清数据） | `localStorage.clear()` | `DELETE FROM user_sessions WHERE user_id=$1` |

### 4.4 user_devices ↔ DevicesTab

| 前端 | DB `user_devices` | 说明 |
|------|-------------------|------|
| `name` | `device_name` | 'Windows 桌面端' |
| `time` | `last_active` | 相对时间格式化 |
| `active` | `is_current` | 当前设备标记 |

### 4.5 user_recent_searches ↔ SearchResultsView

| 前端 | DB `user_recent_searches` | 说明 |
|------|--------------------------|------|
| `recentSearches[i]` | `term` | 搜索词 |
| （无） | `searched_at` | 时间，用于排序/去重 |

### 4.6 membership ↔ MembershipDialog

| 前端常量 `PLANS` | DB `membership_plans` | 说明 |
|------------------|---------------------|------|
| `key` | `plan_key` | 'free' / 'pro' / 'enterprise' |
| `name` | `name` | 套餐名称 |
| `price` | `price_cents` | 分→格式化 |
| `desc` | `description` | 描述文本 |
| `features[]` | `features` (JSONB) | 功能列表 |
| `highlighted` | `highlighted` | 是否推荐 |
