
# 官网首页 - React 组件拆分方案

> **技术栈**: Next.js 16 (App Router) + React 19 + TypeScript + Tailwind CSS 4
> **路径别名**: `@/*` 映射至项目根目录
> **设计系统**: 基于 "Mint Outline" 配色体系（`surface`、`primary`、`outline-variant` 等），Inter 字体排版，Material Symbols 图标库，毛玻璃（glassmorphism）风格。

---

## 一、整体页面架构

### 1.1 页面级组件

| 组件名 | 文件路径 | 类型 | 核心职责 |
|--------|----------|------|---------|
| `HomePage` | `@/app/page.tsx` | 服务端页面（含客户端子组件） | 作为官网首页入口，自上而下组合所有区块组件。 |

### 1.2 组件层级树

```
HomePage                                    // 页面入口 - 组装所有区块
├── Navbar                                  // 导航栏（固定定位、毛玻璃效果）
│   ├── Logo                                // Logo 图标 + 品牌名称
│   ├── NavLinks                            // 桌面端导航链接列表
│   └── CTAButtons                          // 登录按钮 + 免费试用按钮
├── HeroSection                             // Hero 主视觉区块
│   ├── HeroBadge                           // 顶部标签徽章（"认知优先·无干扰书写"）
│   ├── HeroTitle                           // 主标题文字
│   ├── HeroDescription                     // 副标题/描述文字
│   ├── HeroActions                         // CTA 按钮组
│   └── DecorativeDivider                   // 装饰分隔线
├── ProductPreviewSection                   // 产品预览区块
│   ├── ProductPreviewCard                  // 预览卡片（毛玻璃 + 阴影）
│   │   ├── PreviewSidebar                  // 模拟侧边栏（搜索 + 文档列表）
│   │   └── PreviewEditor                   // 模拟编辑器内容区域
│   └── FeatureTags                         // 底部标签组（零干扰编辑器/全局搜索等）
├── FeaturesSection                         // 核心特色网格区块
│   ├── SectionHeader                       // 区块标题 + 副标题
│   └── FeatureCard[]                       // 4 个特色卡片（认知负荷/全局搜索/任务即笔记/加密同步）
├── DesignPhilosophySection                 // 设计哲学双栏区块
│   ├── PhilosophyContent                   // 左侧文本内容区
│   │   ├── SectionBadge                    // "设计哲学"标签
│   │   ├── PhilosophyTitle                 // 标题文字
│   │   ├── PhilosophyDescription           // 描述段落
│   │   └── PhilosophyHighlights[]          // 特性亮点列表
│   └── TestimonialCard                    // 右侧用户证言卡片
├── StatsSection                            // 数据统计区块
│   └── StatItem[]                          // 4 个统计指标（留存率/活跃用户/评分/认知摩擦）
├── CTASection                              // 底部行动召唤区块
│   ├── CTAIcon                             // 装饰图标
│   ├── CTATitle                            // 标题文字
│   ├── CTADescription                      // 描述文字
│   ├── CTAButtonGroup                      // 按钮组
│   └── CTADisclaimer                       // 底部免责提示
└── Footer                                  // 页脚
    ├── FooterBrand                         // Logo + 品牌描述 + 社交图标
    ├── FooterLinkColumn[]                  // 3 列链接（产品/资源/公司）
    └── FooterCopyright                     // 版权声明
```

### 1.3 数据流概览

```
HomePage（组装层，无复杂状态）
  │
  ├── 所有子组件均为"展示组件"
  ├── 数据通过 Props 以静态方式传递
  ├── Navbar 响应滚动事件改变透明度（内部 useScrollEffect 管理）
  └── 按钮点击事件由各组件内部处理（或转发至父组件）
```

## 二、组件详细拆分

### 2.1 Navbar（导航栏）

| 属性 | 内容 |
|------|------|
| **组件名** | `Navbar` |
| **文件路径** | `@/components/landing/navbar.tsx` |
| **类型** | 客户端组件（`'use client'`） |
| **核心职责** | 固定在页面顶部的毛玻璃导航栏。包含品牌 Logo、桌面端导航链接、登录/试用按钮。监听滚动事件动态调整背景透明度和阴影。 |

**包含的 HTML 元素**：
- `<header class="fixed top-0 w-full z-50 glass-nav">` — 固定定位，毛玻璃背景 `rgba(246, 250, 247, 0.72)` + `backdrop-filter: blur(12px)`，底部半透明边框
- 内层 `<div class="max-w-7xl mx-auto px-6 md:px-8 py-4 flex items-center justify-between">` — 最大宽度 1280px 水平居中容器

**子组件**：
- `Logo`（品牌图标 + 名称）
- `NavLinks`（桌面端导航链接）
- `CTAButtons`（登录 + 免费试用）

**Props 接口**：
```typescript
interface NavbarProps {
  navItems?: { label: string; href: string }[];
  onLogin?: () => void;
  onSignup?: () => void;
}
```

**状态与行为**：
- 使用 `useEffect` 监听 `window.scroll` 事件
- 滚动 > 20px 时：增加 `shadow-soft` 阴影，背景透明度从 0.72 变为 0.88
- 使用 `useCallback` + `useState` 管理滚动状态，滚动事件需做防抖/清理

**样式作用域**：
- 容器：`fixed top-0 w-full z-50 glass-nav transition-all`
- 内层：`max-w-7xl mx-auto px-6 md:px-8 py-4 flex items-center justify-between`
- 自定义 class `glass-nav`：`background: rgba(246, 250, 247, 0.72); backdrop-filter: blur(12px); border-bottom: 1px solid rgba(189, 201, 196, 0.25)`

**交互逻辑**：
- 滚动时背景透明度变化
- 内部按钮点击事件回调至父组件

<link href="https://fonts.googleapis.com/css2?family=Inter:opsz,wght@14..32,400;14..32,500;14..32,600;14..32,700;14..32,800&display=swap" rel="stylesheet">
