## 一、产品概述

### 1.1 产品定位

ClarityNotes（心智笔记）是一款面向知识工作者的在线文档管理工具，提供文件树导航、富文本编辑、自动保存、回收站等核心功能，帮助用户高效组织和编辑文档。

### 1.2 版本目标（V1）

V1 为纯前端 MVP，所有数据存储在客户端（内存状态），不使用后端和数据库。

| 目标 | 说明 |
|------|------|
| 完成官网落地页 | 介绍产品核心价值和特性 |
| 完成认证系统 | 登录/注册/退出（Mock 数据） |
| 完成应用主页 | 文件树 + 内容编辑两大核心模块 |
| 完成回收站 | 已删除文档的查看/恢复/彻底删除 |

### M02 - 认证系统

#### 4.2.1 登录

| 项目 | 要求 |
|------|------|
| 登录方式 | 邮箱 + 密码 |
| 表单字段 | 邮箱（email input）、密码（password input） |
| 附加选项 | "记住我" 复选框 |
| 错误提示 | 邮箱格式错误、密码错误、账号不存在（红色提示条） |
| 成功行为 | 跳转至 `/workspace` |
| 社交登录 | Google 登录、Apple 登录（UI 占位，V1 不实现） |
| 切换注册 | 底部 "没有账号？立即注册" 链接 |

#### 4.2.2 注册

| 项目 | 要求 |
|------|------|
| 表单字段 | 姓名、邮箱、密码、确认密码 |
| 校验规则 | 邮箱格式、密码长度 ≥ 6、两次密码一致 |
| 成功行为 | 跳转至 `/workspace` |
| 切换登录 | 底部 "已有账号？立即登录" 链接 |

#### 4.2.3 找回密码（V1 占位）

| 步骤 | 说明 |
|------|------|
| 1. 输入邮箱 | 用户输入注册邮箱 |
| 2. 邮箱验证码 | 输入收到的验证码 |
| 3. 设置新密码 | 输入新密码 + 确认密码 |
| 4. 确认 | 完成重置，返回登录页 |

#### 4.2.4 路由访问控制

| 规则 | 说明 |
|------|------|
| 未登录 | 访问 `/workspace`、`/trash` → 重定向到 `/login` |
| 已登录 | 访问 `/login` → 重定向到 `/workspace` |
| 退出登录 | 清除认证状态，跳转到 `/login` |
| 注销账号 | 确认弹窗 → 清除数据 → 跳转到 `/` |

### M04 - 文件树

#### 4.4.1 树节点类型

| 类型 | 图标 | 说明 |
|------|------|------|
| 文件夹 | `folder`（Material Icon） | 可展开/折叠 |
| 文档 | `description`（Material Icon） | 点击打开编辑视图 |

#### 4.4.2 嵌套层级限制

| 规则 | 值 |
|------|-----|
| 最大嵌套深度 | 5 层 |
| 第 5 层限制 | 只能创建文档，不能创建文件夹 |

示例结构：
```
我的文档（根，第 1 层）
  └─ 项目文件夹（第 2 层）
      └─ 项目文件夹（第 3 层）
          └─ 项目文件夹（第 4 层）
              └─ 项目文件夹（第 5 层）
                  └─ 我的文档.md（只能放文档）
```

#### 4.4.3 节点操作按钮

| 按钮 | 显示条件 | 菜单项 |
|------|---------|--------|
| ＋ 号 | 文件夹节点（层级 < 5） | 新建文件、新建文件夹 |
| ＋ 号 | 文件夹节点（层级 = 5） | 新建文件（无新建文件夹选项） |
| ⋯ 号 | 所有节点 | 重命名、删除 |

#### 4.4.4 新建节点

**场景一：用户不主动重命名**

| 步骤 | 行为 |
|------|------|
| 1 | 以 "未命名" 作为默认名称 |
| 2 | 若同级已存在同类型 "未命名"，自动添加后缀：未命名 (2)、未命名 (3)... |
| 3 | 节点自动进入内联编辑模式，输入框获得焦点 |

**场景二：用户主动输入名称**

| 步骤 | 行为 |
|------|------|
| 1 | 输入框获得焦点，默认值 "未命名" 全选 |
| 2 | 用户输入新名称 |
| 3 | 按 Enter / 失焦 → 保存（触发校验） |
| 4 | 按 Esc → 取消创建（删除节点） |

**名称校验规则**

| 校验项 | 规则 | 错误提示 |
|--------|------|---------|
| 非空 | 名称不能为空或纯空格 | "名称不能为空" |
| 非法字符 | 不能包含 `\ / : * ? " < > \|` | "名称不能包含 \\ / : * ? \" < > \| 等字符" |
| 长度限制 | 不超过 100 个字符 | "名称不能超过 100 个字符" |
| 重名 | 同级目录同类型不能重名 | "同级已存在同名项目" |

#### 4.4.5 重命名

| 项目 | 要求 |
|------|------|
| 触发 | 点击 ⋯ → 重命名 |
| 交互 | 内联编辑模式，输入框替代节点名称 |
| 保存 | Enter / 失焦（校验通过后保存） |
| 取消 | Esc（恢复原名称） |

#### 4.4.6 删除

| 项目 | 要求 |
|------|------|
| 触发 | 点击 ⋯ → 删除 |
| 行为 | 从树中移除节点（当前版本不进入回收站，V1 直接删除） |
| 文件夹删除 | 递归删除所有子节点 |

#### 4.4.7 展开 / 折叠

| 行为 | 说明 |
|------|------|
| 点击箭头 | 切换展开/折叠 |
| 点击节点 | 选中节点，右侧切换到对应视图 |
| 折叠状态 | 隐离子节点，不销毁数据 |

### M06 - 文档编辑视图

#### 4.6.1 触发条件

左侧文件树点击文档节点。

#### 4.6.2 元数据栏

| 字段 | 类型 | 说明 |
|------|------|------|
| 文档标题 | 可编辑 `<h1>` | 失焦保存，同步更新左侧树节点名称；为空时自动设为 "未命名" |
| 创建时间 | 只读 | 格式 `MM/DD HH:mm` |
| 更新时间 | 只读 | 格式 `MM/DD HH:mm` |
| 字数统计 | 实时 | 纯文本字数（不含 HTML 标签） |
| 自动保存状态 | 状态指示器 | "已保存" / "保存中…" |

#### 4.6.3 正文编辑器

**支持格式**

| 格式 | 说明 |
|------|------|
| 段落 | 默认文本 |
| 标题 | H1 / H2 / H3 |
| 无序列表 | `•` 符号 |
| 有序列表 | `1.` `2.` `3.` |
| 引用块 | `<blockquote>` |
| 代码块 | `<pre><code>` |
| 表格 | 插入表格 |
| 图片 | 通过 URL 插入 |

**编辑行为**

| 行为 | 说明 |
|------|------|
| 输入 | 直接键入 |
| 粘贴 | 支持粘贴纯文本和 HTML |
| 删除 | Backspace / Delete |
| 自动保存 | 修改后 2 秒无操作自动保存 |

#### 4.6.4 浮动工具栏（BubbleMenu）

| 触发条件 | 用户用鼠标选中一段文本 |
|------|------|
| 位置 | 选区上方（空间不足时显示在下方） |
| 隐藏条件 | 取消选中 / 点击外部 |

| 按钮 | 功能 |
|------|------|
| **B** 粗体 | 加粗选中文本 |
| *I* 斜体 | 斜体选中文本 |
| ~~S~~ 删除线 | 添加删除线 |
| 🔗 链接 | 插入超链接 |
| `</>` 代码 | 将选中文本包裹为内联代码 |

#### 4.6.5 空白文档

| 触发条件 | 文档正文内容为空，或新建后从未编辑 |
|------|------|
| 占位符 | 灰色斜体："开始撰写你的内容..." |
| 聚焦时 | 占位符消失，正常输入 |
| 清空后失焦 | 重新显示占位符 |
| 标题为空 | 保存时自动设为 "未命名" |

### M07 - 搜索栏

> ⬜ **状态：V1 待实现**

#### 4.7.1 位置

左侧侧边栏顶部，用户头像下方、文件树上方。

#### 4.7.2 搜索范围

| 版本 | 范围 |
|------|------|
| V1 | 仅搜索文档标题（精确匹配 + 模糊匹配，忽略大小写） |
| V2 | 扩展至正文 |

#### 4.7.3 交互逻辑

| 步骤 | 行为 |
|------|------|
| 1 | 用户输入关键词 |
| 2 | 防抖 300ms 后触发搜索，或按回车立即触发 |
| 3 | 左侧文件树自动隐藏，切换为搜索结果视图 |
| 4 | 点击搜索结果 → 右侧加载文档编辑视图 |
| 5 | 点击返回 / 清除 → 恢复文件树 |

#### 4.7.4 搜索结果视图

| 元素 | 说明 |
|------|------|
| 返回按钮 | ← 返回文件树 |
| 结果统计 | "共 X 条搜索结果" |
| 结果项 | 文档标题（高亮匹配词）、文件路径（面包屑）、匹配内容片段（≤50 字） |
| 无结果 | 空状态："未找到相关文档" |

#### 4.7.5 匹配片段规则

从文档正文中截取关键词所在的一句话，最多 50 字符。
